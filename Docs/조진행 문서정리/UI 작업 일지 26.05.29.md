## 피그마 수정사항 반영 (설계도 업데이트 완료)

- 직원 상태 3가지로 구체화 (프로젝트 진행중 / 교육활동 참여중 / 대기중)
- 칭호(HonorificLabel) 추가 — 직원관리 / 직원고용 / 직원해고 / 직원교육 상세페이지
- 과거 참여 프로젝트 이력 — 2줄 표시, 초과 시 세로 스크롤 확정
- CancelHireButton 제거 — 기획서 상 채용취소 버튼 없음 확인
- ConfirmButton 조건 추가 — 보유자금 < 투입비용 시 Canvas_Alert.AlertPopup 호출
- 지원자 없을 경우 비활성화 → Canvas_Alert.AlertPopup 호출 방식으로 변경

# 개요
- 구글시트 URL을 활용한 데이터 연동 사용법에 대한 설명입니다.

## 사용법 요약
1. DataRequestSet SO에셋을 생성
2. 인스펙터에서 시트 URL작성 및 startRow 설정
3. 적용할 Target SO List 연결 (SheetDataSOBase를 상속받은 SO)
4. Target SO List들의 ID를 시트 1열의 ID와 매칭
5. '구글시트 데이터 불러오기' 버튼 클릭하여 데이터 적용

## 코드 작성
### Scripts\Util\GoogleSheet
- SheetData : url을 받아 string형태로 가공. 성공하면 DataRequestSet에서 콜백 실행
- SheetDataSOBase : 시트에서 데이터를 받아올 SO의 추상 클래스. id 기준 및 파싱 관리. 시트 데이터를 받으려면 이것을 상속받아 사용.
### Settings\Editor
- DataRequestSet : SheetDataSOBase를 상속받은 SO 리스트를 시트 1열의 ID 기준으로 매칭하여 데이터 적용
- GoogleSheetEditorMenu : 에디터 상단에 '구글시트' 라는 메뉴 추가. 해당 index의 DataRequestSet가 열림
### WIP\GoogleSheetParseTest
- WebRequestTestSO : SheetDataSOBase를 상속받은 SO. 작성한 코드들을 테스트하기 위한 용도
- 해당 폴더에 DataRequestSet을 포함한 SO들이 테스트 용도로 존재
